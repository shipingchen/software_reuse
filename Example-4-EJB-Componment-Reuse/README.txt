Description
-----------
This is a simple example to show how to reuse code via Object inherence.
The example is programmed in Java.

Prerequisite
----------------
1.	Java 1.7.x or up
2.	Apache-tomee-8.0.2-plus or up
3.	Eclipse IDE for Enterprise Java Developers - Latest Release

Steps
-----
1.	Open Ecllipse IDE
2.	Open the project "Tomee"
3.	Run as server

If everything goes well, you can see a brower within your IDE displying:


Hello from Servlet!

Hello from EJB!


or open a brower with http://localhost:8080/Tomee/HelloServlet, you will see the same as above.
